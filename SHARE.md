# TCG Vault PWA — Share Package

## What this is
A phone-first Pokémon + Riftbound deck builder, value tracker, manual card entry tool, and price lookup hub.

## Open locally on Matt's LAN

```txt
http://192.168.0.7:8191
```

## Install on phone

### iPhone / Safari
1. Open the LAN URL.
2. Tap Share.
3. Tap Add to Home Screen.
4. Name it TCG Vault.

### Android / Chrome
1. Open the LAN URL.
2. Tap menu ⋮.
3. Tap Add to Home screen / Install app.

## What works
- Pokémon live search.
- Deck builder.
- Riftbound watchlist.
- Manual card entry.
- Price lookup tab.
- TCGplayer/eBay/PriceCharting/Magical Meta/Riftbound.gg external links.
- Local-only data storage.
- JSON export/import.

## Honest limitations
- No cloud sync yet.
- No OCR/card camera yet.
- Riftbound pricing is manual/external-check only.
- Affiliate tag field is blank until a real approved affiliate account exists.
